Tips for Choosing a Great Username

  • Keep it Short: The shorter your username, the better it will be for mobile device users.
  • Avoid Numbers: If you append "123" to your name, it may look as if you are a bot or an afterthought.
  • Think Long-Term: You may be okay with your current name now, but what about five or ten years from now? Choose a name that will remain professional in the long term.
  • Act Fast: Good usernames are taken in an instant. If you find a good one, grab it quickly.

Frequently Asked Questions

Why do some checks take longer?

Each platform has its own rate limits and response times. Sites like Twitter and TikTok often have stricter bot detection, which may result in an "Uncertain" status or a slightly longer wait time to ensure accuracy.

How does the username checker work?

If you enter a username that you wish to use and click on "Check Now," we immediately make a request in real-time to the public profile endpoint for all platforms at the same time. If a profile for that username exists, we say that username is "Taken." If no profile for that given username exists, we say that username is "Available." The results are displayed immediately without waiting for all platforms to complete checking.

What is "Uncertain"?

There are times when a profile might be blocking or rate limiting us from making a request to them, and this is particularly true for Instagram, which is notoriously bad about this. If this happens, we won't be able to tell you for sure if the username is taken or not, so we'll display "Uncertain" for you with a yellow dot. If you click on this card, we'll take you to the profile so that you can check for yourself!

Which platforms do you check?

We currently support Instagram, TikTok, Twitter (X), Reddit, and YouTube. We are constantly expanding this list to include more social networks and developer platforms.
