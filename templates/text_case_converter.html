What Is a Text Case Converter?

A text case converter is a tool that transforms your text into different capitalisation styles instantly. Whether you're a developer naming variables, a writer formatting headlines, or a designer polishing UI copy, having the right case format saves time and prevents errors.

What's the difference between camelCase and PascalCase?

camelCase starts with a lowercase letter and capitalises the first letter of each subsequent word: myVariableName. PascalCase (also called UpperCamelCase) capitalises the first letter of every word including the first: MyClassName. In practice: use camelCase for JavaScript variables and functions, PascalCase for classes and React components, TypeScript types and interfaces.

When should I use snake_case vs kebab-case?

snake_case uses underscores as separators: user_profile_data. It's the standard convention in Python (variables, functions, file names) and in SQL column names. kebab-case uses hyphens: user-profile-data. It's used in CSS class names, HTML attributes, URL slugs, and many command-line tools. JavaScript can't use hyphens in variable names, so never camelCase-vs-kebab in JS code directly, only in strings or CSS.

Does Title Case capitalise every word?

Not quite, proper Title Case follows rules about "minor words". Short prepositions (in, on, at, by), conjunctions (and, or, but), and articles (a, an, the) are typically left lowercase unless they're the first or last word of the title. Our converter follows this convention automatically, so "the quick brown fox and the lazy dog" becomes "The Quick Brown Fox and the Lazy Dog".
