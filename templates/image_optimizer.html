Frequently Asked Questions

Which formats do you support?

We support all common image formats including JPEG, PNG, and WebP. You can convert from JPEG/PNG into the highly efficient WebP format for maximum performance gains.

What output format should I choose? JPEG, PNG, or WebP?

JPEG is perfect for photographs, it offers excellent compression with minimal compromise on quality. PNG is perfect for graphics, it is lossless and offers support for transparency. WebP is the new standard format supported by modern browsers, it generally beats JPEG and PNG in output size, but it requires your audience to run a relatively modern browser, which is pretty much everyone in 2026.

Does compression affect quality?

Our optimizer uses lossy compression algorithms tuned for the web. While you may see a slight reduction in absolute resolution, we prioritize visual fidelity so the image looks identical to the human eye while being significantly smaller.

How does this improve SEO?

Page speed is a critical ranking factor for Google. Fast-loading images improve your Largest Contentful Paint (LCP), one of the Core Web Vitals, which can help increase your search engine visibility.

Are there file size limits?

The tool works with most web images, such as photos taken directly from cameras or mobile phones, screenshots, and graphics. Large files above 30MB may take longer to process. If you are optimizing many large, high-quality photos, consider doing them a few at a time.
